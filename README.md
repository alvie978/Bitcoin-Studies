## 📖 Project Overview
This project analyzes Bitcoin market data to understand trends and patterns in cryptocurrency prices. Through data exploration and visualization, the study examines how Bitcoin prices change over time and highlights key insights about market behavior. The project demonstrates how data analysis can be applied to financial and cryptocurrency datasets.

## 💼 Business Problem

Cryptocurrency markets are highly volatile, making it challenging for investors and analysts to understand price trends and market behavior. Data analysis can help uncover patterns and provide insights into how cryptocurrency prices fluctuate over time.

This project aims to explore Bitcoin price data and identify trends within the cryptocurrency market.

## 📊 Files Included
*   **`Bitcoin.ipynb`**: A Jupyter Notebook (created with Google Colab) containing the Python code for data analysis, statistical comparisons, and visualizations.
*   **`Bitcoin dashboard.twbx`**: An interactive Tableau dashboard that visualizes key findings, trends, and comparisons from the dataset.
*   **`Dataset.csv`**: The primary dataset used for this study, sourced from Kaggle. It contains historical price data and relevant trading metrics for Bitcoin and selected traditional stocks/indices.

## 🚀 How to Use This Project
1.  **Explore the Data**: Start by reviewing `Dataset.csv` to understand the raw information.
2.  **Review the Analysis**: Open `Bitcoin.ipynb` in Jupyter Notebook or Google Colab to see the step-by-step analytical process and code.
3.  **Interact with Visuals**: Download the `Bitcoin dashboard.twbx` file and open it with Tableau Public or Tableau Desktop to explore the interactive charts and filters.
